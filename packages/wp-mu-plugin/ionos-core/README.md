# IONOS Core MU-Plugin

- updates itself
- loop functions
- marketplace funtions

## Installation

Run these commands on the remote server.

```
cd wp-content/mu-plugins
curl -O https://s3-eu-central-1.ionoscloud.com/web-hosting/ionos-group/ionos-core/ionos-core.zip
unzip ionos-core.zip
rm ionos-core.zip
```

replace USER and HOST with the data from **waas-create**-output.
